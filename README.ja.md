# chart-bar-plotly

Plotly.jsを使用した棒グラフ表示コンポーネントです。外部データの読み込みや、グラフの向き、サイズ、凡例の表示などのカスタマイズが可能です。

## デモ
[chart-bar-plotly demo](https://code4fukui.github.io/chart-bar-plotly/)

## 機能
- CSV形式の外部データを読み込めます
- グラフの向き、サイズ、凡例の表示などをオプションで指定できます

## 必要環境
- Plotly.js

## 使い方
`<chart-bar>` 要素を使用することで棒グラフを表示できます。データはオブジェクトやCSVから読み込むことができ、オプションでグラフの表示設定ができます。
