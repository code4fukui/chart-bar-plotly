# chart-bar-plotly

1〜2文で説明。Plotly.jsを使用したカスタムHTML要素によるシンプルな棒グラフ表示コンポーネントです。

## デモ
[chart-bar-plotly demo](https://code4fukui.github.io/chart-bar-plotly/)

## 機能
- データを簡単に棒グラフ化できる
- CSV形式の外部データも読み込める
- グラフの向き、サイズ、凡例の表示などをオプションで指定可能

## 必要環境
- Plotly.js

## 使い方
chart-bar.jsをインポートし、`<chart-bar>`タグを使用することで棒グラフを表示できます。
データはオブジェクトやCSVから読み込むことができます。
オプションでグラフの向きやサイズを指定できます。
